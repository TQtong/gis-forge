# GeoForge DevPlayground — GIS 可视化系统界面设计（v2）

> **定位**：功能完整的 Web GIS 应用，验证 GeoForge 引擎全部能力。
> 像 QGIS/ArcGIS Online 一样操作——加载数据、切换视图、绘制标注、空间分析、调整样式。
> v2 修复了 v1 的 27 个设计缺陷（工具栏过载/面板职责混乱/功能缺失/状态缺失）。

## 2. 顶部工具栏（TopToolbar）

高度 48px。左-中-右三区：

### 2.1 左区：品牌 + 视图

| 元素 | 规范 |
|------|------|
| Logo | GeoForge 图标 20px + "GeoForge" 文字 font-semibold。点击 → flyTo 初始位置 |
| 视图切换 | SegmentedControl 3 项：`2D`（正交，pitch=0）/ `2.5D`（透视，可 pitch/bearing）/ `Globe`（地球模式，大气+星空）。切换触发 ViewMorph 动画。**激活项**用 accent 背景 + 白字，其余 ghost |

> **为什么去掉「3D」**：原 v1 有 2D/2.5D/3D/Globe 四个选项。2.5D 和 3D 在用户视角几乎相同（都是透视 + 旋转），区别只是底层投影细节。合并为 `2.5D`（默认 pitch≤85°）。Globe 是明确不同的体验（球面投影 + 大气）。

### 2.2 中区：搜索

```
┌──────────────────────────────────────┐
│ 🔍  搜索地点或输入坐标...              │  ← 宽度 min-w-[280px] max-w-[400px]
└──────────────────────────────────────┘
```

搜索交互详细流程：

| 状态 | UI |
|------|-----|
| 空闲 | placeholder "搜索地点或输入坐标..." |
| 输入中 | 下拉候选列表（最多 8 个）。每项：📍图标 + 地名 + 国家/地区标签。键盘↑↓选择，Enter 确认 |
| 输入坐标格式 | 检测 `116.4, 39.9` 或 `39°54'N 116°23'E` → 显示"飞行到此坐标"选项 |
| 搜索中 | 输入框右侧显示 Spinner |
| 无结果 | 下拉显示"未找到匹配地点" |
| 网络错误 | 下拉显示"搜索服务不可用，请检查网络" |
| 选中结果 | flyTo 目标位置 + 地图上显示临时标记（5 秒后消失）|

### 2.3 右区：编辑工具栏 + 全局操作

**编辑工具栏**是一组互斥的工具按钮（同时只能激活一个）：

```
[🖐 平移] [👆 选择▾] │ [✏️ 绘制▾] [📏 测量▾] [📊 分析▾] │ [⚙️] [📤]
  默认       ↓            ↓          ↓          ↓        设置 导出
           单击选择      点         距离      缓冲区
           框选        折线       面积      叠加分析
           套索        多边形     高程剖面   等值线
                      矩形                 热力密度
                      圆                   Voronoi
                      自由线               视域分析
                                          坡度/坡向
                                          山体阴影
                                          分类统计
```

**工具激活态设计**：
- 默认工具 = 🖐平移（地图 pan/zoom/rotate）
- 点击任何工具 → 该工具按钮高亮（accent 背景）+ 鼠标光标变化
- 再次点击同一工具 或 按 Esc → 回到平移模式
- 激活绘制/测量时，地图上方出现**操作提示条**：

```
┌──────────────────────────────────────────────────────────────┐
│ ✏️ 多边形绘制  │  单击放置顶点 · 双击完成 · Esc 取消 · Ctrl+Z 撤销 │
└──────────────────────────────────────────────────────────────┘
```

**⚙️ 设置按钮**：点击 → 弹出**模态对话框**（不占用右侧面板）。见第 7 节。

**📤 导出按钮**下拉菜单：

```
┌─────────────────────────┐
│ 📷 导出为 PNG 图片       │
│ 📄 导出为 PDF            │
│ 📦 导出所有图层 GeoJSON  │
│ 🔗 复制当前视图链接      │
│ 📋 复制嵌入代码 (iframe)  │
│ ─────────────────────── │
│ 🔖 保存为书签            │ ← 新增
└─────────────────────────┘
```

## 3. 地图视口（MapViewport）

### 3.1 渲染区

- WebGPU Canvas，无边框无圆角，100% 宽高填满
- 鼠标交互：左键 pan / 滚轮 zoom / 右键 rotate（2.5D/Globe）/ Shift+右键 pitch
- 触摸：单指 pan / 双指缩放旋转

### 3.2 地图叠加控件

```
┌────────────────────────────────────────────────────────┐
│                                                        │
│ [操作提示条]（仅激活工具时显示，居中顶部）                 │
│                                                        │
│                                              [+]       │ ← 右上控件组
│                                              [-]       │    缩放
│                                              [⊙]      │    定位
│                                              [🧭]     │    指南针（旋转=bearing）
│                                                        │
│                           [鹰眼图]                      │ ← 右下角 120×90px
│                           ┌─────┐                      │    半透明
│                           │ ▭   │                      │    显示视口位置框
│                           └─────┘                      │
│                                                        │
│ ┌──────────────┐          ├──── 500m ───┤              │ ← 比例尺（右下，鹰眼上方）
│ │📍 116.397°E  │                                       │
│ │   39.908°N   │                                       │ ← 左下坐标信息
│ │   z: 12.3    │                                       │
│ └──────────────┘                                       │
│                                                        │
│ [时间轴]（仅有时序数据时显示，底部横跨）                    │
│ ┌──────────────────────────────────────────────────┐   │
│ │ [▶] ──●────────────────────── [2024-03-15 14:30] │   │
│ │      2024-01   2024-02   2024-03   2024-04       │   │
│ └──────────────────────────────────────────────────┘   │
└────────────────────────────────────────────────────────┘
```

**鹰眼图（Mini-map）**：
- 位置：右下角，尺寸 120×90px，圆角 6px，半透明背景
- 显示简化的全球/区域底图 + 当前视口范围矩形
- 点击鹰眼图任意位置 → 地图 flyTo 到该位置
- 可通过设置关闭

**时间轴控件**（条件显示）：
- 仅当图层列表中存在时序数据（TrajectoryLayer / 带时间字段的 GeoJSON）时自动出现
- 底部横跨地图宽度，高度 48px
- 播放/暂停按钮 + 滑块 + 当前时间标签 + 倍速选择
- 拖动滑块 → 实时更新地图上的时序可视化

**Popup/Tooltip（要素悬停）**：

```
鼠标悬停在地图要素上 → 300ms 延迟后显示浮窗：

┌──────────────────┐
│ 长安街            │  ← 主要属性（text-field 或第一个字符串属性）
│ highway: primary  │  ← 次要属性（前 3 个属性）
│ lanes: 6          │
│ 点击查看详情 →    │  ← 提示文字
└──────────────────┘
   ▼（指向要素的尖角）

点击要素 → Popup 固定显示 + 右侧属性面板展示完整信息
点击地图空白区域 → Popup 消失
```

### 3.3 右键菜单

```
┌───────────────────────┐
│ 📍 在此处添加标注       │
│ 📐 从此处开始测量       │
│ 🔍 查询此处要素 (3)     │  ← 括号内为该位置命中的要素数
│ ──────────────────── │
│ 📋 复制坐标 (116.4, 39.9) │
│ 📋 复制坐标 (度分秒)     │
│ 🗺️ 以此为中心           │
│ 🔖 保存为书签           │  ← 新增
│ 🔗 分享此位置           │
└───────────────────────┘
```

### 3.4 文件拖放反馈

用户拖拽文件到地图区域时：

| 状态 | 视觉反馈 |
|------|---------|
| 拖入边界 | 地图区域出现全屏半透明遮罩（accent色10%透明度）+ 虚线边框 + 中心显示 "📂 释放以加载数据" |
| 悬停中 | 遮罩持续，鼠标显示 copy 光标 |
| 释放 | 遮罩消失 → 自动检测文件格式 → 显示"正在解析 xxx.geojson..." toast → 成功后图层出现在列表 + flyTo 数据范围 |
| 格式不支持 | 红色 toast "不支持的文件格式 .xxx，支持: geojson/json/csv/kml/gpx" |
